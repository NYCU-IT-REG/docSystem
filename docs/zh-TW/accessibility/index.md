# 網站無障礙聲明

本網站依據「無障礙網頁開發規範」（Web Content Accessibility Guidelines, WCAG 2.1）等級 AA 標準進行設計與開發，以確保所有使用者都能平等地獲取資訊。

## 無障礙功能

### 1. 鍵盤導航支援

本網站完全支援鍵盤操作，您可以使用以下快速鍵：

- **Tab 鍵**：移動到下一個可操作元素
- **Shift + Tab**：移動到上一個可操作元素
- **Enter 鍵**：選擇或啟動目前的元素
- **空白鍵**：捲動頁面或勾選核取方塊
- **方向鍵**：在選單或清單中移動

### 2. 跳過區塊

網頁提供「跳至主要內容」連結，讓使用鍵盤或螢幕閱讀器的使用者能快速跳過導航區域，直接進入主要內容。

### 3. 替代文字

所有圖片、圖示和多媒體內容都提供適當的替代文字描述，確保使用螢幕閱讀器的使用者能理解內容。

### 4. 標題結構

網頁使用適當的標題層級（H1 至 H6），建立清楚的內容階層，方便快速瀏覽與導航。

### 5. 色彩對比

文字與背景的對比度符合 WCAG 2.1 AA 標準：
- 一般文字：至少 4.5:1
- 大型文字：至少 3:1
- 圖形元素：至少 3:1

### 6. 深色模式

提供深色模式切換功能，讓使用者依個人需求選擇舒適的閱讀模式。

### 7. 文字縮放

網站使用相對單位設計，支援瀏覽器的文字縮放功能（最高至 200%），不會影響版面配置或功能。

### 8. 表單標籤

所有表單欄位都有明確的標籤說明，並在必要時提供額外的輔助說明。

### 9. 錯誤訊息

表單驗證錯誤會提供清楚的錯誤訊息，說明錯誤原因與修正方法。

### 10. 語言標示

網頁正確標示主要語言（繁體中文或英文），並支援多國語言切換。

## 螢幕閱讀器相容性

本網站經過以下螢幕閱讀器測試：

- **NVDA**（Windows）
- **JAWS**（Windows）
- **VoiceOver**（macOS/iOS）
- **TalkBack**（Android）

## 減少動畫

本網站遵循使用者的系統偏好設定，當偵測到「減少動畫」設定時，會自動減少或停用動畫效果。

## 鍵盤焦點指示

所有可互動元素在獲得鍵盤焦點時，都會顯示清楚的視覺指示，確保使用者知道目前所在位置。
